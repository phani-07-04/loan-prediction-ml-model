# loan-approval-prediction-ml
Loan Approval Prediction using Machine Learning

This project predicts whether a bank loan should be approved based on customer data using machine learning techniques.

Project Features
- Data preprocessing and missing value handling
- Exploratory Data Analysis
- Feature Engineering
- Model training using Random Forest and Gradient Boosting
- Model evaluation using accuracy, ROC-AUC, and confusion matrix

Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib
Seaborn

Machine Learning Models
Random Forest Classifier
Gradient Boosting Classifier

Results
The model was evaluated using cross-validation and ROC-AUC to ensure reliability.
